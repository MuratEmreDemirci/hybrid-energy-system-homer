# ⚡ Hybrid Renewable Energy System Design (HOMER Pro)

This project presents the design and analysis of hybrid renewable energy systems using HOMER Pro for both grid-connected and off-grid scenarios.

---

## 📌 Project Scope

* Grid-connected system design
* Off-grid (standalone) system design
* Load profile modeling (CREST / Demand Ninja)
* Renewable generation (solar & wind)
* Energy storage (battery systems)

---

## ⚙️ Tools Used

* HOMER Pro
* Demand modeling tools (CREST / Demand Ninja)
* Renewable data (Renewables Ninja)

---

## 📊 System Components

* Solar PV
* Wind Turbine
* Battery Storage
* Grid Connection
* Load Demand Profiles

---

## 📈 Analyses Performed

* Cost optimization
* System sizing
* Energy balance
* Grid vs off-grid comparison
* Renewable penetration analysis
